### Run Locally

Clone the project

```bash
  git clone https://github.com/saksham1236/saksham-tandon-capstone-lws-front-end
```

Go to the project directory

```bash
  cd /**folder that you cloned**/
```

Install dependencies

```bash
  npm install
```

Start the React Server

```bash
  npm start
```
